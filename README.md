# Teste_Analytics_LARISSA_CARDOZO
Case técnico – Análise exploratória de dados de vendas
![Dashboard de Vendas](https://github.com/laricscs/Teste_Analytics_LARISSA_CARDOZO/blob/main/dashboard.png)

---


Este repositório contém a resolução de um **case técnico de análise de dados**, com foco em **Python, análise exploratória, visualização de dados e consultas SQL**.  
O objetivo foi analisar um dataset de vendas, identificar padrões relevantes e extrair insights para apoio à tomada de decisão.

## 📁 Estrutura do Projeto

```text
├── dataset.py                          # Script para geração do dataset simulado
├── dados_limpos.csv                    # Dataset após limpeza e tratamento
├── analise_exploratoria_vendas.ipynb   # Análise exploratória e gráfico de tendência mensal
├── analise_vendas_produto.ipynb        # Análise de vendas por produto
├── consultas_sql.sql                   # Consultas SQL solicitadas no case
├── Relatório de Insights e Recomendações.pdf  # Relatório final com insights
├── painel.png                          # Imagem do dashboard/gráfico final
└── README.md                           # Documentação do projeto
```

---
## 📈 Análises Realizadas

### 🔹 Parte 1 – Análise Exploratória

- Cálculo do total de vendas por produto  
- Identificação do produto com maior e menor volume de vendas  
- Análise da evolução das vendas ao longo do tempo (mensal)  

### 🔹 Parte 2 – Visualização de Dados

- Gráfico de linha demonstrando a tendência mensal de vendas em 2023  
- Identificação de padrões e variações ao longo do ano  

### 🔹 Parte 3 – SQL

- Consulta do total de vendas por produto e categoria  
- Identificação dos produtos com menor volume de vendas em junho de 2023  

---

## 💡 Principais Insights

- O **Notebook** foi o produto com maior valor total de vendas no período analisado.  
- O **Mouse** apresentou o menor volume de vendas.  
- Observou-se um pico de vendas no mês de **junho**, seguido por uma queda significativa em **julho**.  

Esses padrões podem indicar **sazonalidade** ou impacto de **campanhas promocionais**.
